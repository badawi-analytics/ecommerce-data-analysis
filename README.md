# E-Commerce Data Analysis Project

## Project Overview
This project analyzes e-commerce orders data to understand customer behavior, delivery performance, and operational issues.

## Datasets Used
- Customers
- Orders
- Order Items
- Payments
- Products
- Sellers
- Geolocation

## Key Questions Answered
- What is the distribution of order statuses?
- Who are the top customers by total spending?
- Which sellers have the highest delivery delays?
- Is there a relationship between product weight and delivery delay?

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Jupyter Notebook

## Key Insights
- Most orders are successfully delivered, but a small percentage face delays or cancellations.
- Some sellers consistently experience higher delivery delays.
- Heavier products tend to have longer delivery times.

## Next Steps
